# Blog Slowpokita

Un blog minimalista sobre desarrollo web, arquitectura de software y diseño moderno construido con arquitectura hexagonal.

## 🚀 Características

- **Arquitectura Hexagonal**: Separación clara de capas (dominio, aplicación, infraestructura, presentación)
- **Next.js 15**: App Router, Server Components, TypeScript
- **Diseño Moderno**: Paleta de colores vibrante (#1E1B4C, #FF2E97, #FFD402, #0ABDC6)
- **Responsive**: Diseño completamente adaptable
- **SEO Optimizado**: Metadatos dinámicos, JSON-LD, sitemap
- **Búsqueda**: Funcionalidad de búsqueda en tiempo real
- **Categorías y Archivo**: Organización por temas y fechas
